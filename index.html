⚙️ Google Drive Setup
One-time setup (5 minutes):
  1. Go to console.cloud.google.com
  2. Create a new project → name it Mahatab Tuition
  3. Go to APIs & Services → Enable APIs
  4. Enable Google Drive API
  5. Go to Credentials → Create OAuth 2.0 Client ID
  6. Type: Web application
  7. Add your app URL to Authorized JavaScript origins
  8. Copy the Client ID
  9. Open your index.html file and replace:
    577695236166-placeholder.apps.googleusercontent.com with your real Client ID
  10. Re-upload to Netlify/GitHub
✅ Once done, the Connect Drive button will work and you can back up to your personal Google Drive.

    Firebase Setup Guide
    5 minutes • Free forever
    💡 Firebase Firestore free tier: 50,000 reads + 20,000 writes per day — more than enough for a tuition centre.
    1
    Go to console.firebase.google.com and sign in with your Google account
    2
    Click "Create a project" → name it ms-tfees → Continue → Disable Google Analytics → Create Project
    3
    In your project, click Build → Firestore Database → Create database → choose Start in test mode → select a location → Enable
    4
    Click the ⚙️ gear icon → Project settings → scroll to "Your apps" → click </> Web → register app as mstfees → Register app
    5
    You'll see a config block. Copy these two values:
    apiKey: "YOUR_API_KEY"
    projectId: "YOUR_PROJECT_ID"
    6
    Paste those values into the Firebase section in Backup → Firebase Cloud Sync and tap Connect Firebase
    ✅ Once connected, use ☁️ Push to Cloud to save data and 📥 Pull from Cloud to restore on any device.
