⚙️ Google Drive Setup
One-time setup (5 minutes):
  1. Go to console.cloud.google.com
  2. Create a new project → name it Mahatab Tuition
  3. Go to APIs & Services → Enable APIs
  4. Enable Google Drive API
  5. Go to Credentials → Create OAuth 2.0 Client ID
  6. Type: Web application
  7. Add your app URL to Authorized JavaScript origins
  8. Copy the Client ID
  9. Open your index.html file and replace:
    577695236166-placeholder.apps.googleusercontent.com with your real Client ID
  10. Re-upload to Netlify/GitHub
✅ Once done, the Connect Drive button will work and you can back up to your personal Google Drive.
